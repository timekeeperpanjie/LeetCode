# LeetCode
个人数据结构与算法练习作业提交仓库

五毒神掌学习法
* 第一遍：读题+思考。
    * 花5-15分钟把解决方法在脑中过一遍，比较时间空间复杂度
    * 没有思路->直接看解法，比较优劣，背诵比较
* 第二遍：自己写
    * leetcode提交->通过：每种解法都通过
    * 比较执行时间
* 第三遍：24h之后再次练习
* 第四遍：一周之后再次练习
    * 专注不熟悉的部分
* 第五遍：恢复性训练
    * 面试之前一周左右开始练习
学习思路：
* 职业训练：拆分知识点->刻意练习->反馈
* 题做多遍：只做一遍没有效果，类比职业训练
* 切题方法：清晰题目要求->分析可能解决路径->写程序->测试通过
